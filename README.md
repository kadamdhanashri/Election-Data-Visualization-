# Election-Data-Visualization-
This Power BI project delivers a multi-level analysis of India’s General Elections from national outcomes to constituency details. Through dynamic charts and reports, complex election data is simplified into actionable insights for better understanding.

**Key Features:-**
**Executive Overview**
National Summary 
Vote Distribution
Total Turnout
